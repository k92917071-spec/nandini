# For Nandini Singh ✨

A personal, privacy-safe static website made with HTML, CSS and JavaScript.

## How to upload on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to **Settings → Pages**.
4. Source: **Deploy from a branch**.
5. Branch: **main**, folder: **/root**.
6. Save and open the generated GitHub Pages link.

## Edit Text

Open `index.html` and change any message, name, or wishes as you like.

## Files

- `index.html` — main website
- `assets/style.css` — design and animation
- `assets/script.js` — interactive surprise effects
